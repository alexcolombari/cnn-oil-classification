# Machine Learning applied to predictive maintenence of industrial equipment.

#### In development

Developed using Deep Learning with Convolutional Neural Networks for image classification applied to microscopic oil samples. This is my final undergraduate project to get Computer Science Bachelor's title.

### Using

##### Requirements

   [Keras](https://keras.io/)<br/>
   [Numpy](https://www.numpy.org/)<br/>
   [Pandas](https://pandas.pydata.org/)<br/>
   [PyCM](https://github.com/sepandhaghighi/pycm)<br/>
   [Scikit-Learn](https://scikit-learn.org/stable/)<br/>
   [TensorFlow](https://www.tensorflow.org/)
  

  Parameters:
  ```
  python oil_class.py EPOCHS DEBUG
  
  EPOCHS -> Number of epochs to train model
  DEBUG:
    1 -> Train Model
    2 -> Test Model
  ```
